# hello-gitub

这是一个用于 GitHub + Codex 跨设备工作的项目仓库。当前目标是先把仓库整理成手机端和电脑端都容易接手的工作区。

## 当前状态

- GitHub 仓库：`fk911cat/hello-gitub`
- 默认分支：`main`
- 本地工作区：`/Users/zoushang/Documents/GitHub项目精选`
- 当前还没有绑定具体技术栈；后续可以按实际项目选择 Web、Python、数据分析或其他开发环境。

## 工作方式

1. 在电脑端或手机端 Codex 中打开同一个 GitHub 仓库。
2. 每次开始前先同步最新代码。
3. 每次完成一段可确认的工作后提交到 GitHub。
4. 如果在多个设备之间切换，先确认前一个设备的改动已经提交并推送。

更多流程见 [`docs/WORKFLOW.md`](docs/WORKFLOW.md)。

## 项目文件

- `AGENTS.md`：给 Codex 的项目工作规则。
- `docs/WORKFLOW.md`：电脑端和手机端协作流程。
- `docs/PROJECT_NOTES.md`：项目背景和后续计划记录。
- `.gitignore`：忽略本地缓存、临时文件和敏感配置。
- `.editorconfig`：统一文本格式。
